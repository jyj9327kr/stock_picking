# Korean Market Quant Trading Pipeline

한국 주식시장(KOSPI / KOSDAQ)에서 대세 상승 국면의 안전한 주도주를 골라내기 위한 파이썬 기반 정량적(Quant) 주식 스크리닝 파이프라인 시스템입니다. 

Mark Minervini의 **8대 추세 템플릿(Trend Template)** 기술적 필터와 실시간 금융감독원 **오픈 DART API** 재무 분석을 결합하여 강력한 모멘텀과 탄탄한 펀더멘탈을 겸비한 "Safe Market Leaders"를 선별합니다.

---

## 🛠️ 주요 특징

1. **Minervini 8대 기술적 필터 직접 적용**:
   * **현재가 위치**: 현재가 > 150일선 및 200일선
   * **이동평균 정배열 1**: 150일선 > 200일선
   * **200일선 상승 추세**: 현재 200일선 > 1개월 전(20영업일 전) 200일선
   * **이동평균 정배열 2**: 50일선 > 150일선 및 200일선
   * **단기 지지 지표**: 현재가 > 50일선
   * **신저가 탈출**: 현재가 >= 52주 신저가 * 1.3 (바닥에서 최소 30% 반등 확인)
   * **신고가 근접**: 현재가 >= 52주 신고가 * 0.75 (고점 대비 25% 이내 유지)
   * **지수 대비 상대 강도**: 개별 종목 3M 수익률 > 시장 기준 3M 수익률
   *(신뢰성 확보를 위해 상장 250영업일 미만의 신규 상장주는 자동 배제됩니다.)*

2. **통합 대세 상승 벤치마크 지수 적용**:
   * KOSPI 지수와 KOSDAQ 지수의 3개월 누적 수익률 중 **더 높게 상승한 지수를 일괄 벤치마크**로 채택합니다.
   * 시장 하락에 편승하여 덜 하락했다는 이유로 통과하는 착시 주식들을 차단하고, 실제 지수 최강자를 능가하는 진정한 모멘텀 리더만 걸러냅니다.

3. **주도 업종 실시간 집계**:
   * Stage 1을 통과한 종목들의 한국거래소(KRX KIND) 업종 분류 데이터를 분석하여, 가장 높은 지분과 강세를 차지하고 있는 **상위 3대 섹터**를 자동으로 추출하여 리포트 상단에 시각화해 줍니다.

4. **오픈 DART API 연계 실시간 재무 검증 (Stage 2)**:
   * 1일 10,000건 제한인 DART 쿼리 낭비를 줄이기 위해, Stage 1을 통과한 선별 종목들에 대해서만 DART 실시간 재무 분석을 적용합니다.
   * 연결재무제표(CFS)와 개별재무제표(OFS)를 안전하게 순회하며 아래 **3대 재무 요건**을 검증합니다:
     * **ROE (수익성)**: 최신 분기 또는 TTM ROE >= 15%
     * **EPS YoY (성장성)**: 전년동기 대비 분기순이익 증가율 >= 15%
     * **EPS QoQ (안정성)**: 직전 분기 대비 순이익 증가율 >= 0% (역성장 배제)

5. **로컬 디스크 캐싱**:
   * 일봉 데이터(1일 유효), 종목 리스트, DART 공시 데이터(30일 유효) 등을 로컬 디스크 캐시(`.pkl`)로 자동 관리하여, 반복 실행 시의 속도를 극대화하고 API 사용량을 보존합니다.

## 🚀 실행 방법

### 1. 환경 설정 및 의존성 설치
`.env` 파일을 루트 디렉토리에 생성하고 발급받은 DART API 키를 지정합니다.
```env
DART_API_KEY="여러분의_오픈_DART_API_키"
```

필요한 패키지를 가상환경 등에 설치합니다:
```bash
pip install -r requirements.txt
```

### 2. 스크리너 파이프라인 구동

```bash
# [매일] 장 마감 후 - Stage 1 (기술적 필터링 및 주도 섹터 3개 추출)
python main.py --schedule daily

# [매주] 주말 - Stage 1 + Stage 2 (실시간 DART 재무 요건까지 통합 검증하여 최종 Watchlist 생성)
python main.py --schedule weekly

# [분기] 실적 공시 시즌 - DART 재무 데이터 대량 업데이트 및 캐싱
python main.py --schedule quarterly
```

* 결과 데이터는 자동으로 `output/` 폴더 하위에 CSV 파일 및 요약 보고서 마크다운(`.md`) 포맷 파일로 타임스탬프와 함께 보관됩니다.
